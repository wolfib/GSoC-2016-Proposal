In the following examples D3.js' force layout was used to visualise short DNA sequences.

Sequence inversions result in a confusing visualisation though. I have changed the edges into arrows to improve the graph’s legibility. The visualisation shows two sequences. Grey arrows symbolise the parts where both sequences are the same. The diverging part is symbolised by red and blue arrows. The subsequence “CTAGCT” is traversed from left to right by the red path and from right to left by the blue path. The way this sequence inversion is displayed by the force field is not elegant and still needs to be improved.

These graphics only represent my first experiments in applying force fields to visualise DNA seqences. It is already obvious that some further modifications would be necessary to achieve better results (better alignment of the nodes, uniform link lenghts,...). A lot of further investigation work needs to be done first, before a decision can be made on whether and how to use (modified) force fields or a different approach for the creation of a JavaScript module for the visualisation of genomic sequence graphs.
