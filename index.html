These graphics only represent my first experiments in applying force fields to visualise DNA seqences. It is already obvious that some further modifications would be necessary to achieve better results (better alignment of the nodes, uniform link lenghts,...). A lot of further investigation work needs to be done first, before a decision can be made on whether and how to use (modified) force fields or a different approach for the creation of a JavaScript DNA graph visualisation module.
